# IAP-Hub

IAP Infrastruktur-Dashboard (Asklepios). Struktur:

- Prod/ - alles fuer den produktiven Betrieb (Dashboard-HTML, Backend-API, Konfiguration)
- Dev/ - Entwicklungs-/Import-Hilfsmittel (Rohdaten-Exporte, Generator-Skripte)
- NeedForService/ - Dateien speziell fuer die Einrichtung als Windows-Taskplaner-Aufgabe
